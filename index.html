Viewerの使い方 / How to Use Viewer

👆 長押しでメニュー表示 / Long Press for Menu

スマホの画面を長押し(約1秒)すると、設定メニューが表示されます。カメラ選択、Tally枠のサイズ変更、Preview表示のON/OFFが切り替えられます。

Long press (about 1 sec) on the screen to show the settings menu. You can select camera, change tally border size, and toggle preview display.

📹 カメラ選択 / Camera Selection

メニューからCAM 1~8を選択すると、そのカメラのTally状態(赤/緑の枠)が表示されます。「None」を選択すると枠表示をOFFにできます。

Select CAM 1-8 from the menu to display that camera's tally state (red/green border). Select "None" to turn off the border.

🎨 Tally枠のサイズ変更 / Border Size

Small (4px) / Medium (8px) / Large (12px) / X-Large (16px)の4段階から選択可能。見やすいサイズに調整してください。

Choose from 4 sizes: Small (4px) / Medium (8px) / Large (12px) / X-Large (16px). Adjust to your preference.

🟢 Preview表示の切り替え / Toggle Preview

「Green ON」のチェックを外すと、Preview(緑枠)を非表示にできます。Program(赤枠)だけを表示したい場合に便利です。

Uncheck "Green ON" to hide Preview (green border). Useful when you only want to see Program (red border).

📺 映像OFFモード / Video Off Mode

Setting Pageで「Video Display Enabled」をOFFにすると、映像は表示されずTally表示のみが画面全体に表示されます。帯域を節約したい場合や、Tallyだけ確認したい時に便利です。

When "Video Display Enabled" is turned OFF on the Setting Page, only the tally display fills the entire screen without video. Useful for saving bandwidth or when you only need tally information.
